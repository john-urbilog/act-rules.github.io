Visible

Key: visible

Content perceivable through sight.

Content is considered visible if making it fully transparent would result in a difference in the pixels rendered for any part of the document that is currently within the viewport or can be brought into the viewport via scrolling.

Content is defined in WCAG.

Contribute






GitHub

Contributing is open to anyone. We welcome any new issues or pull requests for changes. ACT Rules Community website has conference calls every 4 weeks. If you are interested in becoming an active contributor or reviewer, we ask that you join the ACT Rules Community website community group through the W3C Website. This requires setting up a W3C account, may require approval by the organization you work for if they are a W3C member.
